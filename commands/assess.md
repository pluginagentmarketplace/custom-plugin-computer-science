# /assess

Self-assess your knowledge level across different domains and get personalized recommendations. This command helps you identify strengths, weaknesses, and learning gaps.

## Usage

```
/assess [domain]
/assess [domain] detailed
/assess all
```

## Examples

```
/assess frontend
Quick assessment of frontend development skills

/assess backend detailed
Comprehensive backend skills assessment with detailed feedback

/assess devops
DevOps and infrastructure skills evaluation

/assess all
Full assessment across all major domains
```

## Assessment Results

Each assessment provides:
- **Current Level**: Where you stand now
- **Strengths**: Areas of expertise
- **Gaps**: Skills to improve
- **Recommendations**: Personalized learning plan
- **Next Steps**: Immediate actions
- **Resources**: Curated materials
- **Timeline**: Expected learning duration
