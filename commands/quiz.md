# /quiz

Test your Computer Science knowledge with interactive quizzes across all major domains.

## Usage

```
/quiz [domain] [difficulty]
/quiz foundations easy
/quiz algorithms medium
/quiz data-structures hard
/quiz systems mixed
/quiz all comprehensive
```

## Difficulty Levels

### Easy (Warm-up)
- Recall facts
- Basic concepts
- Simple calculations
- Foundation check

**Example**:
What is the time complexity of binary search?
a) O(n)
b) O(log n)
c) O(n²)
d) O(1)

### Medium (Assessment)
- Apply concepts
- Compare approaches
- Solve problems
- Deeper understanding

**Example**:
Which sorting algorithm has O(n) worst-case time complexity?
a) Quicksort
b) Merge sort
c) Bubble sort
d) Heap sort

### Hard (Challenge)
- Complex analysis
- Theoretical knowledge
- Multi-concept problems
- Advanced topics

**Example**:
Prove that comparison-based sorting requires Ω(n log n) comparisons.

## Quiz Format

### Multiple Choice
