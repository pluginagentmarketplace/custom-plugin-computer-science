# /skills

View required skills, competencies, and learning resources for any developer role. This command helps you understand what you need to learn for your target position.

## Usage

```
/skills [role-name]
/skills [role-name] [specialization]
/skills
```

## Examples

```
/skills frontend
Shows all frontend development skills

/skills backend python
Shows backend development skills with Python focus

/skills devops kubernetes
Shows DevOps skills for Kubernetes

/skills data-science ai
Shows data science and AI skills

/skills security
Shows cybersecurity required skills
```

## Learning Resources

Each skill includes:
- **Official Documentation**: Language & framework docs
- **Online Courses**: Structured learning (Udemy, Coursera, etc.)
- **Interactive Tutorials**: Hands-on practice
- **Books**: Deep dives and theory
- **Projects**: Real-world applications
- **Communities**: Forums and Discord servers
- **Podcasts**: Industry insights
- **Blogs**: Latest trends and tips

## Skill Validation

For each skill, you can:
- Self-assess your current level
- Find gaps in knowledge
- Get recommendations for improvement
- Track progress over time
- Earn certifications
- Participate in assessments

## Career Progression

Skills are organized by role and progression:
1. **Entry Level**: Basic competencies needed
2. **Mid Level**: Intermediate skills and specialization
3. **Senior Level**: Advanced expertise and leadership
4. **Architect/Lead**: Strategic thinking and system design
