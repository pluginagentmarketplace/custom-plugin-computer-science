# /learning-path

Get a personalized learning path based on your current role, target career, and available time. This command helps you create a structured plan to reach your development goals.

## Usage

```
/learning-path [current-role] [target-role] [timeframe]
/learning-path
```

## Examples

```
/learning-path frontend react
Create a path from frontend to React specialization

/learning-path backend microservices 6-months
Learn microservices architecture in 6 months

/learning-path beginner python 3-months
Start Python from scratch in 3 months

/learning-path qa automation fast-track
Quick path to test automation

/learning-path
Interactive mode to define your learning path
```

## Supported Roles

### Level 1: Beginner
- HTML & CSS Fundamentals
- JavaScript Basics
- Python Basics
- Git & Version Control

### Level 2: Intermediate
- **Frontend**: React, Vue, or Angular
- **Backend**: Node.js, Python, or Java
- **DevOps**: Docker fundamentals
- **Data**: SQL and Pandas

### Level 3: Advanced
- **Full Stack**: Frontend + Backend integration
- **Architect**: System design and patterns
- **DevOps**: Kubernetes and cloud platforms
- **AI/ML**: Machine learning models
- **Security**: Application and infrastructure security

### Level 4: Specialist
- **React Expert**: Advanced patterns and optimization
- **Backend Architect**: Microservices and distributed systems
- **Platform Engineer**: Infrastructure and automation
- **ML Engineer**: Production ML systems
- **Security Specialist**: Penetration testing and compliance

## Features

- **Current Assessment**: Evaluate your starting point
- **Goal Setting**: Define your target role
- **Time Estimation**: Realistic timeframes for learning
- **Milestone Planning**: Breaking down into achievable steps
- **Resource Recommendations**: Curated learning materials
- **Progress Tracking**: Monitor your advancement
- **Skill Validation**: Assess skill acquisition

## Learning Paths Available

### Frontend Specialist Path (4-6 months)
1. HTML & CSS Fundamentals (2 weeks)
2. JavaScript Deep Dive (4 weeks)
3. TypeScript Basics (2 weeks)
4. React or Vue (6 weeks)
5. Projects & Portfolio (4 weeks)

### Backend Developer Path (4-6 months)
1. Programming Language Basics (2 weeks)
2. Databases & SQL (3 weeks)
3. Web Frameworks (4 weeks)
4. REST API Design (2 weeks)
5. Authentication & Security (2 weeks)

### DevOps Engineer Path (6-8 months)
1. Linux Fundamentals (3 weeks)
2. Docker & Containerization (3 weeks)
3. Kubernetes Orchestration (4 weeks)
4. Cloud Platform (AWS/GCP) (4 weeks)
5. Infrastructure as Code (3 weeks)

### Data Science Path (6-9 months)
1. Python & Statistics (4 weeks)
2. Data Analysis & Pandas (3 weeks)
3. Machine Learning Basics (5 weeks)
4. Deep Learning (4 weeks)
5. ML Projects (4 weeks)

### Full Stack Developer Path (9-12 months)
1. Frontend Basics (4 weeks)
2. Backend Fundamentals (4 weeks)
3. Databases (2 weeks)
4. Frontend Framework (6 weeks)
5. Backend Framework (6 weeks)
6. Deployment & DevOps (3 weeks)
7. Capstone Project (4 weeks)

## Customization

Each learning path can be customized by:
- Adjusting pace (fast-track, standard, leisurely)
- Selecting preferred technologies
- Focusing on specific domains
- Including project-based learning
- Adding specializations

## Progress Tracking

The learning path includes:
- Weekly milestones
- Skill checkpoints
- Project deliverables
- Assessment criteria
- Recommended resources
