# /roadmap

Browse and explore 65+ developer roadmaps from the developer-roadmap repository. Find the perfect learning path for any tech role or specialization.

## Usage

Use this command to discover and explore roadmaps organized by category:

```
/roadmap frontend
/roadmap backend
/roadmap devops
/roadmap data-science
/roadmap languages
/roadmap architecture
/roadmap security
/roadmap [search-term]
```
