# 5-Minute Quick Start

Get up and running with the Computer Science Plugin in minutes.

## Step 1: Install (30 seconds)

In Claude Code:
```
/plugin add ./custom-plugin-computer-science
```

Or:
```
/plugin add ~/.claude-code/plugins/computer-science
```

## Step 2: Choose Your Path (1 minute)

```
/learn foundations      ‚Üê Start with math & logic (4 weeks)
/learn algorithms       ‚Üê Master problem solving (8 weeks)
/learn full-path        ‚Üê Complete education (24 weeks)
```

**First-timer?** Run `/learn foundations`

## Step 3: Solve Problems (3 minutes)

```
/problem arrays easy    ‚Üê Find easy array problems
/problem two-sum hint   ‚Üê Get hints for two-sum problem
/problem two-sum solution ‚Üê See the solution explained
```

## Step 4: Test Knowledge (30 seconds)

```
/quiz foundations easy  ‚Üê Take easy foundations quiz
/quiz all comprehensive ‚Üê Full assessment across all domains
```

## Step 5: Analyze Performance (30 seconds)

```
/analyze merge-sort     ‚Üê Understand merge sort complexity
/analyze compare binary linear ‚Üê Compare algorithms
```

## Struggling?
